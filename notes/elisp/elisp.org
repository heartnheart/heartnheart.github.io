* 常用函数速查
** =nth=
#+BEGIN_SRC emacs-lisp
  (nth N LIST)
#+END_SRC
返回 =LIST= 的第 =N= 个元素。 =N= 从 =0= 开始。若 =N= 大于 =LIST= 的长度，则返回 =nil= 。 如果 =N= 是一个负数，则返回 =LIST= 的第一个元素。
#+begin_src emacs-lisp
(nth n x) == (car (nthcdr n x))
#+end_src
Example:
#+BEGIN_SRC emacs-lisp
(nth 0 '(a b c)) ;==> a
(nth 2 '(a b c)) ;==> c
(nth 4 '(a b c)) ;==> nil
(nth -1 '(a b c)) ;==> a
#+END_SRC
