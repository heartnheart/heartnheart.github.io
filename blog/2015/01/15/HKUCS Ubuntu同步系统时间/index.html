HKUCS Ubuntu同步系统时间

默认情况下，同步时间的活不用我们操心，因为Ubuntu会在启动的时候调用一次 ntpdate 同步时间1。 但是由于NTP协议本身可能被利用进行DDoS攻击2，港大CS内网无法正常使用外网的NTP服务器。不过CS内网有自己的NTP服务器3可用来解决这个问题。

使用 ntpd 实时同步时间

  1. 安装 ntpd 
    sudo apt-get install ntp
  2. 修改默认时间服务器：打开 /etc/ntp.conf 文件，在 #Specify one or more NTP Servers. 下面添加 
    # Specify one or more NTP servers
server ntp.cs.hku.hk
